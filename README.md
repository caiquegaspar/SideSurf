# SideSurf 🔲

[![Firefox Add-on](https://img.shields.io/badge/Firefox-Addon-FF7139?logo=firefox-browser&logoColor=white)](#)
[![License: MIT](https://img.shields.io/badge/License-MIT-blue.svg)](https://opensource.org/licenses/MIT)

_For the English version, [click here](#english-version)._

O **SideSurf** é uma extensão avançada para Firefox que adiciona um mini-navegador totalmente funcional na barra lateral.

Diferente do split view nativo, o painel permanece visível independentemente da aba ativa, permitindo multitarefa real sem perder contexto. Ideal para documentação, dashboards, redes sociais ou visualizações paralelas.

## 🔥 Funcionalidades

- **Navegação Completa:** Barra de URL com auto-completar, histórico interno e reload.
- **Múltiplas Abas:** Até 6 abas independentes dentro da sidebar.
- **Layouts Flexíveis:**  
  `⊡ Tabs` (uma por vez) ou `⊟ Tiling` (grade inteligente).
- **Compatibilidade Aprimorada:** Melhor suporte para sites complexos dentro do painel lateral.
- **Controle de Links:** Navegação mantida dentro da extensão.
- **Persistência Automática:** Abas, zoom e histórico são salvos.
- **i18n & Tema Dinâmico:** Suporte a múltiplos idiomas e integração com Dark/Light mode.

## 🚀 Atalhos

| Atalho              | Ação                  |
| ------------------- | --------------------- |
| `Ctrl+Shift+S`      | Abrir/fechar SideSurf |
| `Ctrl+L`            | Focar barra de URL    |
| `Ctrl+T` / `Ctrl+W` | Nova / fechar aba     |
| `Alt+←` / `Alt+→`   | Navegação             |

# SideSurf (English) 🇬🇧

**SideSurf** is an advanced Firefox extension that adds a fully functional mini-browser to your sidebar.

Unlike Firefox’s native split view, the sidebar remains visible across all tabs, enabling true multitasking without losing context. Perfect for documentation, dashboards, or secondary browsing.

## 🔥 Features

- **Full Navigation:** URL bar with autocomplete, history, and reload.
- **Multi-Tab Support:** Up to 6 independent tabs inside the sidebar.
- **Flexible Layouts:** Tabs mode or smart tiling grid.
- **Enhanced Compatibility:** Improved support for complex websites inside embedded views.
- **Link Control:** Keeps navigation within the extension.
- **Automatic Persistence:** Tabs, zoom, and history are saved.
- **i18n & Theming:** Multi-language support with automatic dark/light mode.
